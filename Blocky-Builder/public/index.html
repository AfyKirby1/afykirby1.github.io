About Blocky Builder

Blocky Builder is a powerful visual world editor designed for game developers and world builders.

Features:

  • 🎨 Visual tile-based editor
  • 🧙 Advanced NPC management
  • 🎮 Game integration tools
  • ⚡ Real-time preview
  • 🎯 Spawn point system
  • 📊 Analytics and statistics

Version: 0.07 - Modular Architecture

Quick Start Guide

Getting Started:

  1. Click "Let's Build!" to open the editor
  2. Select a tile from the palette
  3. Click and drag to paint your world
  4. Use tools to add NPCs, spawn points, and more
  5. Export your world when ready!

Keyboard Shortcuts:

  • D - Draw tool
  • E - Erase tool
  • N - NPC tool
  • S - Spawn tool
  • Tab - Cycle tools
